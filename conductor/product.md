# Product Guide

## Overview
This project sets up a "lite mode" of the [easi-remote-access](https://github.com/jeronimo-mg/easi-remote-access) tool on this Linux instance.

## Target Audience
- **Myself (Personal):** The system is primarily designed for personal remote access and management.

## Core Goals
- **Quick web desktop:** The primary objective is to provide a basic, fast remote desktop accessible directly via a web browser.

## Key Features
- **noVNC Web Client:** Web-based VNC client for browser access.
- **Cloudflare Tunnel:** Secure tunneling to expose the service without port forwarding.
- **x11vnc Server:** A simple and fast VNC server for Linux.

## Desktop Environment
- **XFCE:** A lightweight and fast desktop environment to ensure minimal resource usage.
